\section{Digit Sum}
\subsection*{题意}
求 $\{1,2,\ldots,k\}$ 中有多少是 $d$ 的倍数。
\subsection*{数据范围}
\begin{itemize}
\item $1 \leq k < 10^{10000}$
\item $1 \leq d \leq 100$
\end{itemize}

\subsection*{题解}

这题的套路是\textbf{\text{数位DP}}。这类题目的基本框架就是求某一个范围$[l,r]$内有多少个数满足 XXX 性质。由于 $[l,r] = [0,r+1) - [0,l)$，我们通常转化为只有上界，而下界为$0$的两个独立 case 分别计算。于是以下只考虑 $[0,r)$ 的计算方法。
